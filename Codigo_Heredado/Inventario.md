# 📦 Sistema de Inventario Inteligente

[![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)](https://www.python.org/)
[![Flask](https://img.shields.io/badge/Flask-2.0+-green.svg)](https://flask.palletsprojects.com/)
[![License](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

Sistema de análisis de inventario que utiliza **Visión Artificial** (simulada) y **Machine Learning** para predecir la demanda y optimizar el stock.

## 🚀 Características

- 🔍 **Detección simulada de productos** mediante visión artificial
- 📊 **Análisis de inventario** en tiempo real
- 🔮 **Predicción de demanda** usando historial de ventas
- 💡 **Recomendaciones inteligentes** para reposición de stock
- 📈 **Dashboard interactivo** con métricas clave

## 📋 Requisitos Previos

- Python 3.10 o superior
- pip (gestor de paquetes de Python)

## 🛠️ Instalación y Ejecución

### 1. Clonar o descargar el repositorio

```bash


1. Instalar dependencias
bash
pip install flask
Nota: Si usas requirements.txt:

bash
pip install -r requirements.txt
2. Ejecutar la aplicación
bash
python app.py
3. Abrir en el navegador (cada archivo utiliza un localhost diferente)
Accede a: http://localhost:5002 o similares

```
